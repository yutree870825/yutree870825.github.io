開始製作網站(一) - 下載Hexo + 部署至Github

2022-10-12

事前準備

要安裝的套件主要有三個:

  1. Node.js
  2. Git
  3. Hexo

接下來，詳細說一下安裝步驟。

Node.js 下載安裝

進入下載頁面https://nodejs.org/zh-tw/download/，並根據自己的裝置選擇下載。

通常會選擇安裝LTS，因為比較穩定。或是選擇其他版本也可以，但是 hexo 官方文件的需求是 nodejs 的版本不能低於 8.10，並建議安裝 10.0 以上的版本。

然後就根據視窗的指示一路安裝就完成了。
(windows用戶要確認Add to Path是否已經勾選，默認為勾選，勾選後我們才能在 cmd 中使用指令)。

若要測試是否安裝成功，可以打開cmd輸入

如果顯示的是你安裝的版本的話，就代表安裝成功啦！

Git 下載安裝

一樣進入官方下載頁面https://git-scm.com/downloads，選擇相應的裝置選項。

以windows為例，點擊框起來的連結就可以下載。

接下來就安照指示一步一步安裝。

如果要測試是否安裝成功，可以打開cmd輸入

如果出現版本號就代表成功安裝了！

Hexo 下載安裝

安裝nodejs的用處就在這裡！下載 hexo 需要用到 npm 套件，確認 nodejs 下載成功後，在 cmd 中輸入指令。

安裝完後，一樣可以在cmd中輸入以下指令來確認是否安裝完成。

如果出現的是一連串的版本號就代表成功安裝。

Github 設定

進入 github 官網 https://github.com/，並註冊一個帳號。

註冊登入完後，點擊右上角的 + 號，並選擇 New repository。

填寫 Respository name，並根據需求選擇選項(可以保持預設就好)，完成後點擊 Create repository 按鈕。
